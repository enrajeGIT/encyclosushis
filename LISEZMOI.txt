寿司ネタ図鑑 — Encyclopédie des poissons à sushi
=================================================

DÉMARRAGE
---------
Ouvrez simplement index.html dans un navigateur. Aucun serveur nécessaire.
Pour mettre en ligne : déposez tout le dossier tel quel sur GitHub Pages,
Netlify, ou n'importe quel hébergement statique.

AJOUTER VOS IMAGES
------------------
Deux dossiers :
  images/poissons/   → photo du poisson entier (Wikipédia, libre de droits)
  images/nigiris/    → photo du nigiri (votre collection)

Le nom de fichier doit correspondre au champ "id" de data.js, au format .jpg :
  images/poissons/maguro.jpg
  images/nigiris/maguro.jpg

Tant qu'une image manque, le site affiche automatiquement le kanji de
l'espèce sur fond indigo — vous pouvez donc remplir au fur et à mesure.
Format conseillé : ratio 4:3, environ 800×600 px.

AJOUTER / MODIFIER UNE ESPÈCE
-----------------------------
Tout le contenu est dans data.js : copiez un bloc { ... }, changez les
champs, et placez les images correspondantes. C'est tout.
